## 简体中文

### 项目简介
**Hello World** 是一款基于 Hysteria 2 协议开发的安卓 VPN 工具。不同于传统 VPN 的“掩耳盗铃”，它通过整套身份伪装技术，让流量在网络审查中完全隐形。本项目开源以接受公众监督，确保作为安全工具的透明性。

### 核心特性
1.  **无法察觉的隐秘能力** 传统 VPN 像遮挡号牌的汽车，虽难辨底细但极易被重点监控。Hello World 拒绝“此地无银三百两”，从头到尾打造整套虚假身份（号牌、驾照、合法目的地），让你的流量看起来像是访问正规公开网站，实现真正的隐身。
2.  **全球网络畅通** 专为网络受限区域（如中国内地）设计，解除封锁，无限制访问全球服务。
3.  **极致的网络可用性** 我们不追求空载下的虚高数字，而是专注于极端网络环境下的“凿穿”能力。在不可靠网络中提供稳定的专线体验（默认上行 100Mbps / 下行 40Mbps），支持 8K 视频无压力播放。
4.  **极致简洁与易用** - **零配置**：无多余选项，无广告，不索取无关权限。
    - **一键连接**：打开即用，支持桌面下拉快捷开关。

## English

### About The Project
**Hello World** is an Android VPN application built on the Hysteria 2 protocol. Unlike traditional VPNs that attempt to hide traffic through obvious encryption, Hello World employs a complete "identity camouflage" strategy to remain invisible to network inspection. This repository is public to ensure the transparency and security expected of a privacy tool.

### Core Features
1.  **Undetectable Stealth Capabilities** Traditional VPNs are like cars with covered license plates—suspicious and prone to monitoring. Hello World provides a full set of "fake credentials" (plates, license, legal destination), making your traffic appear as legitimate visits to public websites. You won't just be hidden; you'll be perceived as normal.
2.  **Global Content Access** Bypass geo-restrictions and network censorship (e.g., in Mainland China) to access any global service or website without limits.
3.  **Optimized Network Resilience** Instead of chasing inflated "peak speed" numbers, we focus on penetrating unreliable networks. It carves out a stable dedicated line (100Mbps Up / 40Mbps Down by default), ensuring smooth 8K YouTube streaming even in harsh network conditions.
4.  **Minimalist & User-Friendly** - **Zero Configuration**: No redundant settings, no ads, and no unnecessary permissions.
    - **One-Tap Connection**: Simple toggle switch with support for Android Quick Settings (Tile service).
